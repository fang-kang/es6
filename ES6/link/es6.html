# ES6

# 1. 变量

ES6中提供了两个声明变量的关键字：const和let

# let的使用

ES6 新增了let命令，用来声明变量。它的用法类似于var。

  • let声明的变量,多了一个块级作用域
{
  let a = 10;
  var b = 1;
}

a // ReferenceError: a is not defined.
b // 1
  • 不存在变量提升
// let 的情况
console.log(bar); // 报错ReferenceError
let bar = 2;
  • 不允许重复声明
let a = 10;
let a = 1;//报错 Identifier 'a' has already been declared

# const的使用

const声明一个只读的常量。常量：值不可以改变的量

  • const声明的量不可以改变
const PI = 3.1415;
PI = 3; //报错
  • const声明的变量必须赋值
const num;
  • 如果const声明了一个对象，仅仅保证地址不变
const obj = {name:'zs'};
obj.age = 18;//正确
obj = {};//报错
  • 其他用法和let一样
1. 只能在当前代码块中使用
2. 不会提升
3. 不能重复

# let与const的使用场景

1. 如果声明的变量不需要改变，那么使用const
2. 如果声明的变量需要改变，那么用let
3. 学了const和let之后，尽量别用var

# 2. 解构赋值

# 数组解构

以前，为变量赋值，只能直接指定值。


let arr = ['张三', '李四', '王五', '赵六']
let zs = arr[0]
let ls = arr[1]
let ww = arr[2]
let zl = arr[3]

console.log(zs, ls, ww, zl)

ES6 允许写成下面这样。

let arr = ['张三', '李四', '王五', '赵六']

let [zs, ls, ww, zl] = arr
console.log(zs, ls, ww, zl)

# 对象解构

解构不仅可以用于数组，还可以用于对象。

let { foo, bar } = { foo: 'aaa', bar: 'bbb' };
foo // "aaa"
bar // "bbb"

给属性名起个别名。

let { foo: baz } = { foo: 'aaa', bar: 'bbb' };
baz // "aaa"

let obj = { first: 'hello', last: 'world' };
let { first: f, last: l } = obj;
f // 'hello'
l // 'world'

函数的参数也可以使用解构赋值。

function test({ id, name, age }) {}

let user = {
    id: 1,
    name: 'zs',
    age: 28,
}
test(user)

# 3. 字符串

# 模版字符串

传统的 JavaScript 语言，输出模板通常是这样写的

let str = 
  'There are <b>' + obj.username + '</b> ' +
  'items in your basket, ' +
  '<em>' + obj.password +
  '</em> are on sale!'

上面这种写法相当繁琐不方便，ES6 引入了模板字符串解决这个问题。

let str = `
  There are <b>${obj.username}</b> items
   in your basket, <em>${obj.password}</em>
  are on sale!`

字符串模版的优点

  • 允许换行
  • 可以使用插值 ${}

# 字符串方法

  • includes()：返回布尔值，表示字符串是否包含参数字符串。
  • startsWith()：返回布尔值，表示原字符串是否以参数字符头部。
  • endsWith()：返回布尔值，表示原字符串是否以参数字符结尾

# 4. ES6新增的两个数组方法

# 1. find

  • find是ES6新增的语法

  • 作用 : 遍历数组, 找到符合条件的一个元素

  • 返回值 : 数组的一个元素 (找不到 返回 undefined)

 /**
  * 需求1 : 找到id=2 的元素对象
  * 需求2 : 如果有多个条件满足的,只获取第一个满足的
  */
 var obj = list.find(function (item) {
    return item.id == 2
 })

# 2. findIndex

  • findIndex是ES6新增的语法

  • 作用 : 遍历数组, 找到符合条件元素的索引

  • 返回值 : 一个索引(找不到 返回 -1)

 /**
  * 需求1 : 找到 done=true的索引
  */
 var index = list.findIndex(function (item) {
  //  return item.done === true
   return item.done
 })
 console.log(index);

# 5. 函数-箭头函数

ES6标准新增了一种新的函数：Arrow Function（箭头函数）。

为什么叫Arrow Function？因为它的定义用的就是一个箭头：

# 基本使用

  • 具名函数： 转换成函数表达式， 右边变匿名函数。

  • 匿名函数： 删掉function, () 后面加个 =>

function fn (x, y) {
    console.log(x + y);
}
||
let fn = function(x, y) {
    console.log(x + y);
}

相当于
//语法： (参数列表) => {函数体}
let fn = (x, y) => {
    console.log(x + y);
}

# 参数详解

  • 如果没有参数列表，使用()表示参数列表
let sum = () => {
    console.log('哈哈')
};
// 等同于：
let sum = function() {    
    console.log('哈哈')
};
  • 如果只有一个参数，可以省略()
// 等同于：
let sum = function(n1) {    
    console.log('哈哈')
};

let sum = n1 => {
    console.log('哈哈')
};
  • 如果有多个参数，需要使用()把参数列表括起来
let sum = function(n1, n2) {    
    console.log('哈哈')
};

let sum = (n1, n2) => {
    console.log('哈哈')
};

# 函数体(返回值)详解

  • 如果箭头函数的代码块部分多于一条语句，就要使用大括号将它们括起来
let sum = n1 => {
    console.log('哈哈')
    console.log('哈哈')
};
  • 如果函数体只有一行一句，并且需要返回这个值，那么可以省略{}和return
    • 有没有返回值,都会返回,有值返回就得到值,没有值返回,默认是 undefined
let fn = function(n1, n2) {
    return n1 + n2;
}

let fn = (n1, n2) => n1 + n2;

# 案例

  1. 有一个数组[1,3,5,7,9,2,4,6,8,10],请对数组进行排序
let arr = [1, 3, 5, 7, 9, 2, 4, 6, 8, 10]
arr.sort((a, b) => a - b)
arr.sort(function(a, b) {
  return a - b
})
  1. 有一个数组['a','ccc','dddd','bb'],请按照字符串长度对数组进行排序
let arr = ['a', 'ccc', 'dddd', 'bb']
arr.sort((a, b) => a.length - b.length)
arr.sort(function(a, b) {
  return a.length - b.length
})
  1. 有一个数组，[57,88,99,100,33,77],请保留60分以上的成绩，返回一个新的数组
let arr = [57, 88, 99, 100, 33, 77]
let newArr = arr.filter(item => item >= 60)
let newArr = arr.filter(function(item) {
  return item >= 60
})

# 9. key和value的读取变量

# 读取属性值

  • 用点属性名的方式获取属性值的时候，点后面不能跟变量，只能跟确切的属性名

  • 我们还可以通过方括号的方式来获取属性值，方括号中是字符串格式的属性名，也可以接收变量

    let obj = {name: 'zs', age: 38}
let myKey = 'name'
console.log(obj[myKey])

# 写入属性值

  • 可以在定义对象的时候通过方括号的形式，给属性名传递一个变量进来。

  • 可以用方括号的形式给对象的属性进行赋值，方括号内可以传递一个变量进来。

    let obj = {name: 'zs', age: 38}
let myKey = 'name'

let obj1 = {
	[myKey]: 'ls',
	age: 44
}

obj1[myKey] = 'ls'
